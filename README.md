# sales_prediction_ml_model
Train a model using data set to predict sales and variations

Data set : Kaggle 
Method : 
Summary (direct answer)
Approach: exploratory data analysis (EDA) + descriptive stats + visualization, then targeted models/analyses:
Influence of age/gender: logistic/regression + stratified summaries + effect sizes
Time patterns: time-series aggregation + decomposition + change-point detection
Category appeal: rank by frequency and revenue; per-customer preference modeling
Relationships age–spending–preferences: correlations + regression + partial dependence
Seasonal shopping adaptation: seasonal decomposition, per-segment comparisons
Purchasing behavior by items-per-transaction: segmentation + hypothesis tests
Price distributions per category: box/violin plots and summary statistics
Deliverable included: a self-contained Python script/notebook scaffold with functions to run all analyses and produce plots and tables.

Exports results (CSV/PNG) for reports

Tools used : python, pycharm, github-copilot
